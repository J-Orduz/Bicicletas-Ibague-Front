# Bicicletas-Ibague-Front

# Run App
Navigate to bicibague-frontend and run `npm install` to get node modules

And run `npm run dev` to start the project in developer mode

# Installed dependencies
react-router-dom@latest, path, -D sass-embedded, leaflet react-leaflet, react-icons --save, i18next react-i18next i18next-browser-languagedetector
